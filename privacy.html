Privacy policy

Last updated: July 6, 2026

What this site does

meter is a static dashboard that visualizes activity data from the clockwork CLI. It runs entirely in your browser.

Data you load

If you use Load .json or drag a file onto the page, that file is read locally by your browser. It is never uploaded to any server.

Data this site collects

None. meter has no analytics, no tracking pixels, no cookies, and no server that receives your data. The published dashboard data (if any) is a static file committed to the repository — it is public by nature of GitHub Pages.

Third-party services

GitHub Pages serves this site. GitHub may log standard request metadata (IP address, user-agent, referrer). See GitHub's privacy statement.

Google Fonts is loaded from Google's CDN to render the typefaces used on this page. Google may log your IP address when the font files are fetched. See Google's privacy policy. No cookies are set by this request.

Cookies

This site sets no cookies.
