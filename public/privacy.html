Overview

meter is a static personal career showcase that visualizes activity data from the clockwork CLI. It runs entirely in your browser. The data shown by default is synthetic sample data generated at build time — it describes no real person or project. There are no analytics, no tracking pixels, no cookies, no non-essential browser storage, no automated decision-making or profiling, and no advertising or marketing. The source code is public, so every claim here can be verified.
