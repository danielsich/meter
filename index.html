How to use meter

This dashboard shows synthetic sample data. To view your own activity, load a clockwork export — it stays in your browser and is never uploaded.

  1. Install the CLI: 
    pip install clockwork-cli
  2. Generate an export: 
    clockwork both export > clockwork-data.json
    Use claude or codex instead of both to scope to a single tool.
  3. Click Load .json above (or drag the file onto the page) to view it here.

Privacy: your file is read locally with your browser's File API. It never leaves your device — nothing is sent to this site, to GitHub, or anywhere else. See the privacy policy.
